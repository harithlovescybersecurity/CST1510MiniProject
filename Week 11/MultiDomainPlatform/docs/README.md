# Multi-Domain Intelligence Platform

## Project Overview
A comprehensive security intelligence platform for managing cybersecurity incidents, IT tickets, and datasets across multiple domains with AI-powered assistance. Built with Streamlit and Python.

## Features
### Authentication System
- Role-based access control (Admin, Analyst, User)
- Secure password hashing with bcrypt
- Session management
- User registration and login

### Multi-Domain Dashboard
- Cyber Security Domain: Incident tracking and management
- Data Domain: Dataset metadata and analytics
- IT Domain: IT support ticket management

### Incident Management
- Create, Read, Update, Delete (CRUD) operations
- Severity classification (Low, Medium, High, Critical)
- Detailed incident descriptions

### Analytics & Visualization
- Bar charts for incident analysis
- Domain-specific metrics
- Real-time data visualization
- Export capabilities

### AI Assistant
- Google Gemini-powered chat interface
- Cybersecurity guidance and analysis

### Administration
- User management interface
- Role assignment and updates
- System monitoring
- Data management
